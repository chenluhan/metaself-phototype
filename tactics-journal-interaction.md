# 战术日志干预模块交互文档 (PRD)

## 1. 概述
本文档定义了“战术日志”模块中两项核心干预功能（“3分钟呼吸法”与“轻度拉伸”）的二级页面交互逻辑、UI 状态及数据展示规则。

## 2. 交互入口
- **位置**: 首页 -> 战术日志 (Tactics Journal) -> 横向滑动卡片 (Intervention Carousel)
- **触发方式**: 点击对应卡片，触发全屏覆盖的二级详情页 (Detail View Overlay)。
- **动画**: 详情页从底部向上滑出 (Slide-up animation)，背景带有模糊遮罩 (Backdrop blur)。

---

## 3. 核心模块一：3 分钟呼吸法 (BreatheDetailView)

### 3.1 页面结构
页面分为上下两部分：
- **上半部分 (60%)**: 沉浸式执行区 (Immersive Execution Area)
- **下半部分 (40%)**: 个人效能审计区 (ROI Data Audit Area)

### 3.2 沉浸式执行区交互
- **核心视觉**: 居中的呼吸光环 (Breathing Halo)。
- **动画逻辑**:
  - **吸气 (Inhale)**: 光环放大 (`scale: 1.5`)，透明度增加 (`opacity: 1`)，持续 4 秒。
  - **呼气 (Exhale)**: 光环缩小 (`scale: 1`)，透明度降低 (`opacity: 0.6`)，持续 6 秒。
- **触觉反馈 (Haptics)**:
  - 吸气开始时：单次短振动 (`navigator.vibrate(50)`)。
  - 呼气开始时：三次短促振动 (`navigator.vibrate([30, 50, 30])`)。
- **倒计时**: 初始 3:00，每秒递减，格式为 `MM:SS`。
- **状态文本**: 静态提示语“跟随手机震动，维持深度呼气。”
- **退出操作**: 点击底部圆形关闭按钮，关闭当前视图，返回首页。

### 3.3 个人效能审计区展示
- **出现动画**: 页面加载后延迟 0.2 秒从底部滑入 (`y: 50 -> 0`, `opacity: 0 -> 1`)。
- **数据展示 (Bento Grid)**:
  - **平均心率降幅**: `-12 bpm` (绿色高亮)
  - **算力折算挽回**: `+5 %` (绿色高亮)
  - **执行完赛率**: `98 %` (白色)
- **AI 诊断文本**: 底部文本框提供基于历史数据的个性化洞察（如：“历史数据显示，降频呼吸能为您有效阻断交感神经亢奋...”）。

## 4. 核心模块二：轻度拉伸 (StretchDetailView)

### 4.1 页面结构
与呼吸法保持一致，分为上下两部分：
- **上半部分 (60%)**: 沉浸式执行区 (Immersive Execution Area)
- **下半部分 (40%)**: 物理干预收益区 (Physical ROI Area)

### 4.2 沉浸式执行区交互
- **拉伸阶段**: 共 3 分钟，分为 3 个阶段，每阶段 60 秒。
  1. **颈部拉伸**: 缓慢转动头部，感受颈部肌肉的延展。
  2. **肩部环绕**: 双肩向上向后画圆，释放肩颈压力。
  3. **躯干扭转**: 保持骨盆稳定，上半身向两侧轻柔扭转。
- **核心视觉**: 旋转的同心圆环，带有绿色发光指示点，持续匀速旋转 (`rotate: 360`, `duration: 60s`)。
- **状态切换**:
  - 根据总倒计时 (`timeLeft`) 自动计算当前所处阶段。
  - 阶段切换时触发触觉反馈 (`navigator.vibrate([50, 50, 50])`)。
- **倒计时**: 显示当前阶段的剩余时间（60秒倒数），格式为 `MM:SS`。
- **进度指示器**: 底部三个圆点，当前阶段对应的圆点高亮显示（绿色发光），其余为半透明白色。
- **退出操作**: 点击底部圆形关闭按钮，关闭当前视图，返回首页。

### 4.3 物理干预收益区展示
- **出现动画**: 页面加载后延迟 0.2 秒从底部滑入。
- **数据展示 (Bento Grid)**:
  - **局部肌肉张力**: `-24 %` (绿色高亮)
  - **微循环血流量**: `+15 %` (蓝色高亮)
- **AI 诊断文本**: 底部文本提供基于久坐状态的干预建议（如：“系统监测到您已持续久坐超过 90 分钟，当前轻度拉伸可有效阻断肌肉僵硬累积...”）。

## 5. 异常处理与边界条件
- **设备兼容性**: 触觉反馈 (Vibration API) 需检测设备是否支持 (`if (navigator.vibrate)`)，不支持的设备仅提供视觉反馈。
- **生命周期**: 组件卸载时必须清除所有 `setInterval` 和 `setTimeout`，防止内存泄漏。
- **倒计时结束**: 倒计时归零时停止计时器，目前设计为停留在 `00:00` 状态，用户需手动点击关闭按钮退出。
