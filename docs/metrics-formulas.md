# 核心指标计算公式文档

## 1. 夜间充能指数 (Morning Readiness)

**定义**: 评估用户经过一夜睡眠后的身体恢复状态，决定了当天的初始精力上限。

**计算公式**:
`夜间充能指数 = (睡眠结构得分 * 40%) + (自主神经恢复得分 * 40%) + (环境应激扣分 * 20%)`

**参数说明**:
- **睡眠结构得分 (0-100)**: 基于深睡比例和REM比例计算。
  - `得分 = (深睡时长 / 总睡眠时长) * 100 * 权重A + (REM时长 / 总睡眠时长) * 100 * 权重B`
  - 理想状态：深睡占比 15-25%，REM占比 20-25%。
- **自主神经恢复得分 (0-100)**: 基于夜间平均 HRV (心率变异性) 与个人基线对比。
  - `得分 = (夜间平均HRV / 个人30天平均HRV) * 100` (最高100)
- **环境应激扣分 (-100-0)**: 睡眠期间的干扰因素（如环境噪音、温度异常、频繁觉醒）。
  - `扣分 = - (觉醒次数 * 5 + 噪音超标时长(分钟) * 0.5)`

**示例**:
- 睡眠结构得分: 85
- 自主神经恢复得分: 70
- 环境应激扣分: -10
- `夜间充能指数 = (85 * 0.4) + (70 * 0.4) + (-10) = 34 + 28 - 10 = 52` (满分100)

## 2. 算力折算挽回 (Cognitive Bandwidth Recovery)

**定义**: 评估物理干预（如3分钟呼吸法）对恢复大脑认知带宽的有效性。

**计算公式**:
`算力折算挽回(%) = (干预后HRV - 干预前HRV) / 个人基线HRV * 100% * 认知映射系数`

**参数说明**:
- **干预前/后HRV**: 干预开始前5分钟和结束后5分钟的平均心率变异性。
- **个人基线HRV**: 用户在平静状态下的长期平均HRV。
- **认知映射系数**: 经验常数（通常取 0.8-1.2），用于将生理指标(HRV)映射到认知表现上。

**示例**:
- 干预前HRV: 30ms
- 干预后HRV: 45ms
- 个人基线HRV: 50ms
- 认知映射系数: 1.0
- `算力折算挽回 = (45 - 30) / 50 * 100% * 1.0 = 30%`

## 3. 局部肌肉张力 (Local Muscle Tension)

**定义**: 评估轻度拉伸等物理干预对缓解肌肉僵硬的程度。

**计算公式**:
`局部肌肉张力变化(%) = (干预后肌电信号均方根值 - 干预前肌电信号均方根值) / 干预前肌电信号均方根值 * 100%`

**参数说明**:
- **肌电信号均方根值 (sEMG RMS)**: 通过可穿戴设备或预估模型得出的肌肉紧张度指标。
- 负值表示张力下降（肌肉放松），正值表示张力上升。

**示例**:
- 干预前RMS: 120 μV
- 干预后RMS: 90 μV
- `局部肌肉张力变化 = (90 - 120) / 120 * 100% = -25%` (表示张力下降25%)

## 4. 微循环血流量 (Microcirculation Blood Flow)

**定义**: 评估干预手段对局部血液循环的促进作用。

**计算公式**:
`微循环血流量变化(%) = (干预后局部皮温变化量 * 转换系数) + (心率变化量 * 权重)`

**参数说明**:
- **局部皮温变化量**: 干预前后皮肤温度的差值。
- **转换系数**: 将温度变化映射为血流量变化的经验值。

**示例**:
- 由于缺乏直接的血流量传感器，通常使用皮温和心率作为替代指标进行估算。
- 假设算法计算得出干预后血流量提升了 15%，则显示 `+15%`。

## 5. 异常情况表示方法

当传感器数据缺失或计算异常时，UI 应作如下处理：

- **数据采集中**: 显示加载动画或 `--`。
- **设备未佩戴/连接断开**: 显示灰色图标并提示“设备未连接”。
- **数据不足无法计算**: 显示 `N/A` (Not Available) 或 `数据不足`，并提供 Tooltip 解释原因（如：“需要至少3天的基线数据”）。
- **极值/异常值**: 如果计算结果超出合理生理范围（如心率降幅超过 50bpm），应触发数据清洗逻辑，UI 上可显示 `需校准` 或暂时隐藏该指标。
